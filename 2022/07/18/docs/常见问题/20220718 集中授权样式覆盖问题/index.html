集中授权样式覆盖问题

一、问题复现

问题样式

正常样式

二、产生原因(问题分析)

fox-ui 升级，公共样式发生变化。导致旧版本功能无法正常显示

二 、解决方案

1、所有子项目style 标签必须添加 scoped 标记

<style scoped></style>

2、样式覆盖

<style scoped>
    .wrapper-out .fox-group .el-radio {
        display: inline-block !important;
    }

    .wrapper-out  /deep/ .fox-group .el-radio .el-radio__input {
        display: inline-block !important;
    }
</style>
