缺少jQuery问题

一、问题复现

二、产生原因(问题分析)

缺少jQuery插件

三、解决方案

1、安装jQuery依赖

npm install jquery --save

2、修改 webpack.base.js 文件

const webpack = require('webpack')

module.exports = {
    plugins: [
        new webpack.ProvidePlugin({
            $: 'jquery',
            jQuery: 'jquery',
            'windows.jQuery': 'jquery'
        })
    ],
}
